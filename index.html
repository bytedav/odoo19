01

Instalar Git y Clonar Addons

Se requiere -u root para tener permisos de instalación. El proceso puede tardar 2-3 minutos.
Comando Terminal
docker exec -u root {WEB} bash -c "apt-get update && apt-get install -y git && git clone {REPO} /tmp/odoo19 && cp -r /tmp/odoo19/* /mnt/extra-addons/ && chown -R odoo:odoo /mnt/extra-addons && echo 'MODULOS_LISTOS'"
02

Generar odoo.conf

Asegura que el addons_path incluya tanto los módulos base como los de Enterprise.
Configuración
docker exec -u root {WEB} bash -c "mkdir -p /etc/odoo && cat > /etc/odoo/odoo.conf << 'EOF' [options] addons_path = /usr/lib/python3/dist-packages/odoo/addons,/mnt/extra-addons db_host = {HOST} db_port = 5432 db_user = {USER} db_password = {PASS} admin_passwd = bytedav EOF"
03

Aplicar Parche AI (Bug stopPropagation)

Crítico en Odoo 19: Evita que el sistema colapse al intentar usar el panel lateral de Inteligencia Artificial.
Patch JS
docker exec -u root {WEB} bash -c "cat > /mnt/extra-addons/ai/static/src/discuss/composer_patch.js << 'EOF' import { Composer } from \"@mail/core/common/composer\"; import { patch } from \"@web/core/utils/patch\"; patch(Composer.prototype, { saveContent() { if (this.thread?.channel_type === \"ai_chat\") return; super.saveContent(); }, onFocusin(ev) { if (ev) super.onFocusin(ev); if (this.thread?.channel_type === \"ai_chat\" && ev?.target) ev.target.select(); }, get wysiwygConfig() { const config = super.wysiwygConfig; return { ...config, getRecordInfo: () => ({ resModel: this.thread?.model, resId: this.thread?.id }) }; } }); EOF"
04

Habilitar IA (pgvector)

Instala el motor de vectores en Postgres para que los Agentes de IA puedan leer documentos.

Instalación pgvector
docker exec -u root {DB} bash -c "apt-get update && apt-get install -y postgresql-17-pgvector" && docker restart {WEB}
05

Ajustes Finales en Base de Datos

Ejecutar después de crear la base de datos en el asistente web.
SQL Post-Deploy
docker exec {DB} psql -U {USER} -d {DBNAME} -c " INSERT INTO ir_config_parameter (key, value) VALUES ('database.expiration_date', '2099-12-31'), ('database.expiration_reason', 'enterprise'), ('web.base.url.freeze', 'True') ON CONFLICT (key) DO UPDATE SET value = EXCLUDED.value; UPDATE res_users SET odoobot_state = 'disabled'; CREATE EXTENSION IF NOT EXISTS vector;"
06

Dependencias de Módulos (PIP)

Necesario para módulos de Redes Sociales, WhatsApp y Marketing Automático.
PIP Install
docker exec -u root {WEB} bash -c "pip3 install google-auth phonenumbers --break-system-packages" && docker restart {WEB}
07

Quitar Marca "Powered by Odoo"

SQL Branding
docker exec {DB} psql -U {USER} -d {DBNAME} -c "UPDATE ir_ui_view SET active = false WHERE name LIKE '%brand_promotion%';"
